# dotfiles

My personal Linux desktop configuration.

> Arch Linux • Hyprland • Quickshell • Illogical Impulse • HyprDock

---

## Desktop

- **OS:** Arch Linux
- **WM:** Hyprland
- **Shell:** Fish
- **Panel:** Quickshell (Illogical Impulse)
- **Dock:** HyprDock
- **Display Manager:** SDDM
- **Terminal:** Kitty
- **File Manager:** Nemo
- **Editor:** VS Code
- **Theme:** WhiteSur Dark
- **Cursor:** Bibata Modern Classic
- **Icons:** WhiteSur
- **GTK:** WhiteSur Dark

---

## Features

- macOS-inspired glass aesthetic
- Rounded dock with pinned applications
- Glass top panel
- Dynamic wallpaper-based theming
- Quickshell widgets
- HyprDock launcher
- WhiteSur icon theme
- Custom SDDM login screen
- Fish shell configuration
- Kitty terminal configuration
